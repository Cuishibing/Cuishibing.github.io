使用GitHub Page可以搭载一个静态页面的博客，但是有个缺点是无法在线更改，在有一些想法的时候我们必须在本地修改文件并push到远程仓库（很不方便） 

我们可以使用GitHub提供的API实现自动提交更改到仓库，即使用API模拟一次提交代码：

API地址：Git database - GitHub Docs

使用API之前需要了解下Git内部Object模型：Git - Git Objects (git-scm.com)
主要下面几个对象：
- commit
- blob
- tree
- ref

可以直接参考这个文件：https://raw.githubusercontent.com/Cuishibing/Cuishibing.github.io/master/js/github.js
